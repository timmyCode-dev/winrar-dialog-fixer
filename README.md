# WinRAR Dialog Fixer & Interface Optimizer 📦🚀

![Platform](https://img.shields.io/badge/platform-Windows-0078D4?style=for-the-badge&logo=windows)

A smart, lightweight background utility designed to eliminate workflow interruptions in WinRAR. By monitoring active window handles, it automatically dismisses evaluation prompts, register reminders, and paywall dialogue screens to ensure a seamless, non-intrusive archiving experience.

## ✨ Features

- **Instant Dialog Suppression:** Uses high-speed win32 GUI hooking to detect and close evaluation screens in less than 5ms.
- **Zero-Resource Background Service:** Runs quietly in the system tray consuming less than 4MB of RAM.
- **Smart Window Tracking:** Matches process IDs accurately so it never interferes with your actual archive structures or password prompts.
- **Auto-Boot Integration:** Optional registry hook to start up alongside Windows for set-and-forget operation.
